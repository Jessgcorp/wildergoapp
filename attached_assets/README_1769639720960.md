# 🌤️ Animated Weather System for WilderGo

A comprehensive React Native weather animation system with real-time route weather alerts and timeline visualization.

## 📦 What's Included

### 1. **AnimatedWeatherIcons.tsx**
Animated weather icons for 7 different conditions:
- ☀️ **Sunny** - Rotating sun animation
- ☁️ **Cloudy** - Floating cloud motion
- 🌧️ **Rainy** - Pulsing rain effect
- ❄️ **Snowy** - Gentle falling snow
- ⛈️ **Stormy** - Intense shaking motion
- 💨 **Windy** - Swaying animation
- 🌫️ **Foggy** - Fade in/out effect

### 2. **WeatherChangeAlert.tsx**
Smart alert system that shows:
- Current vs. upcoming weather conditions
- Distance until weather change
- Severity levels (low/medium/high)
- Multiple alerts with auto-rotation
- Slide-in animations

### 3. **RouteWeatherTimeline.tsx**
Horizontal scrolling timeline displaying:
- Weather conditions along entire route
- Temperature at each point
- Time and location information
- Distance markers
- Smooth card animations

### 4. **INTEGRATION_GUIDE.tsx**
Complete examples and API integration patterns

## 🚀 Quick Start

### Installation

1. Copy all component files to your project's components folder
2. Install dependencies (if not already installed):
```bash
npm install @expo/vector-icons
```

### Basic Usage

```tsx
import { AnimatedWeatherIcon } from './components/AnimatedWeatherIcons';
import { WeatherChangeAlert } from './components/WeatherChangeAlert';
import { RouteWeatherTimeline } from './components/RouteWeatherTimeline';

// In your Map screen component:
<View>
  {/* Simple weather icon */}
  <AnimatedWeatherIcon 
    condition="rainy" 
    size={48} 
    color="#FF6B35" 
  />

  {/* Weather alerts */}
  <WeatherChangeAlert
    weatherChanges={[
      {
        location: 'Denver, CO',
        distance: '45 miles',
        currentCondition: 'sunny',
        upcomingCondition: 'stormy',
        severity: 'high'
      }
    ]}
    visible={true}
  />

  {/* Route timeline */}
  <RouteWeatherTimeline
    weatherPoints={[
      {
        time: '2:00 PM',
        location: 'Moab, UT',
        condition: 'sunny',
        temperature: 75,
        distance: 'Start'
      },
      // ... more points
    ]}
  />
</View>
```

## 📍 Integration with Map Tab

Add to your existing Map screen:

```tsx
export const MapScreen = () => {
  const [weatherAlerts, setWeatherAlerts] = useState([]);
  const [routeWeather, setRouteWeather] = useState([]);

  useEffect(() => {
    // Fetch weather data
    const fetchWeather = async () => {
      const data = await getRouteWeather(currentRoute);
      setWeatherAlerts(data.alerts);
      setRouteWeather(data.timeline);
    };

    fetchWeather();
    const interval = setInterval(fetchWeather, 300000); // 5 min
    return () => clearInterval(interval);
  }, [currentRoute]);

  return (
    <View style={{ flex: 1 }}>
      {/* Your existing map */}
      <MapView ... />

      {/* Add weather alerts on top */}
      <WeatherChangeAlert
        weatherChanges={weatherAlerts}
        visible={weatherAlerts.length > 0}
      />

      {/* Add timeline at bottom */}
      <View style={{ position: 'absolute', bottom: 100, left: 0, right: 0 }}>
        <RouteWeatherTimeline weatherPoints={routeWeather} />
      </View>
    </View>
  );
};
```

## 🔌 Weather API Integration

Connect to your preferred weather service:

```tsx
const getRouteWeather = async (routeCoordinates) => {
  // Example using OpenWeatherMap or similar
  const response = await fetch(
    `https://api.weather.com/route-weather?coords=${routeCoordinates}`
  );
  const data = await response.json();
  
  return {
    alerts: data.changes.map(change => ({
      location: change.location,
      distance: `${change.miles} miles`,
      currentCondition: mapWeatherCode(change.from),
      upcomingCondition: mapWeatherCode(change.to),
      severity: change.severity
    })),
    timeline: data.forecast.map(point => ({
      time: formatTime(point.timestamp),
      location: point.location,
      condition: mapWeatherCode(point.weatherCode),
      temperature: point.temp,
      distance: `${point.milesFromStart} mi`
    }))
  };
};

// Helper to map API codes to our conditions
const mapWeatherCode = (code) => {
  const mapping = {
    '01d': 'sunny',
    '02d': 'cloudy',
    '09d': 'rainy',
    '13d': 'snowy',
    '11d': 'stormy',
    '50d': 'foggy',
    // ... add your API's weather codes
  };
  return mapping[code] || 'cloudy';
};
```

## 🔧 Troubleshooting

### Icons not showing
Make sure `@expo/vector-icons` is installed:
```bash
npm install @expo/vector-icons
```

### Fonts not loading
Add to your `app.json`:
```json
"expo": {
  "fonts": [
    "./assets/fonts/RussoOne-Regular.ttf",
    "./assets/fonts/Outfit-Regular.ttf"
  ]
}
```

### Animations laggy
Use `useNativeDriver: true` (already implemented) and reduce animation complexity if needed.
